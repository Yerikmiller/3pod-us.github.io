How to use

Step 1: Include JS.

<script src="./iconfont.js"></script>

Step 2: Use CSS.

<style>
.icon {
  width: 1em;
  height: 1em;
  vertical-align: -0.15em;
  fill: currentColor;
  overflow: hidden;
}
</style>

Step 3: Use icon like this.

<svg class="icon" aria-hidden="true">
  <use xlink:href="#icon-xxx"></use>
</svg>
